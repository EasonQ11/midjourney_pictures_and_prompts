# midjourney_pictures_and_prompts

这是用爬虫从midjourney上下载下来的评分较高的图片，和整理成xls格式的图片信息，包括图片的关键词prompt、图片得分hot_score，作者id、作者姓名等。
初使用midjourney的小伙伴可以利用这些prompt来生成自己的图片，不用再烦恼该怎么写词了。咒语已经生成，少年，魔法世界在等着你。
